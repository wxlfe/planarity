# planarity

## about

### goal

solve as many graphs as you can each day with as few moves as possible.

### why every puzzle is solvable

every graph in planarity is solvable because it's a planar graph. a planar graph
is a graph that can be drawn on a flat surface so that no edges cross, except
where they meet at shared vertices.

[wikipedia: planar graph](https://en.wikipedia.org/wiki/Planar_graph)
